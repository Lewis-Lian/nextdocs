## 项目构成

**Markdown/MDX:** Markdown 是一种用于创建格式化文本的标记语言。 MDX 是 Markdown 的扩展，允许在 Markdown 文件中嵌入 JSX 组件。

**[Bun](https://bun.sh):** JavaScript 运行时，我们用它来运行脚本。

**[Node.js](https://nodejs.org):** JavaScript 运行时，我们用它来运行脚本。

**[pnpm](https://pnpm.io):** 快速、节省磁盘空间的包管理器。

## 基于Fumadocs的文档主题制作的文档站存储仓库

本项目基于 [Fumadocs](https://fumadocs.dev) 构建，Fumadocs 是一个现代化的文档网站生成器，支持多种前端框架和内容源，提供丰富的功能和灵活的定制选项。
