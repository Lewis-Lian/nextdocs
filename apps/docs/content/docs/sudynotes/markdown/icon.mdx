## 介绍

本项目集成了lucide的图标库

可点击此链接查询图标库：[https://lucide.dev/icons/](https://lucide.dev/icons/)

## mdx文件头图标使用方法

创建mdx文档时，图标库的图标名称需要首字母大写，并去除图标库名称的`-`符号。

例如添加图标名为`clipboard-pen-line`的图标：

```mdx title="abc.mdx"
---
title: 文档页标题
description: 该文档的简介
icon: ClipboardPenLine
---
```
